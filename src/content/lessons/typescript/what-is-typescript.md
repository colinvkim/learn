TypeScript is JavaScript with a static type system layered on top.

The goal is not to make code academic. The goal is to make intent, assumptions, and mistakes visible earlier.

In practice, that gives you:

- Better editor feedback while you write code.
- Safer refactors across large codebases.
- Clearer public APIs for functions, components, and libraries.
