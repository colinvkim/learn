Once you can move around the filesystem, the next step is changing it.

## Create directories

<ShellCommandSet
  title="Create a directory"
  unix={`mkdir notes
mkdir -p src/content/lessons`}
  powershell={`New-Item -ItemType Directory notes
New-Item -ItemType Directory src/content/lessons -Force`}
  cmd={`mkdir notes
mkdir src\\content\\lessons`}
/>

The exact flags differ, but the task is the same: create a directory at the path you specify.

## Create an empty file

<ShellCommandSet
  title="Create a file"
  intro="Unix shells often use a dedicated file command. Windows shells use different names, but the same basic idea applies."
  unix={`touch README.md`}
  powershell={`New-Item -ItemType File README.md`}
  cmd={`type nul > README.md`}
/>

## Copy files or directories

<ShellCommandSet
  title="Copy content"
  unix={`cp README.md README.backup.md
cp -r src src-backup`}
  powershell={`Copy-Item README.md README.backup.md
Copy-Item src src-backup -Recurse`}
  cmd={`copy README.md README.backup.md
xcopy src src-backup /E /I`}
/>

## Move or rename files and directories

<ShellCommandSet
  title="Move or rename"
  unix={`mv notes.txt docs/notes.txt
mv draft.txt final.txt`}
  powershell={`Move-Item notes.txt docs/notes.txt
Move-Item draft.txt final.txt`}
  cmd={`move notes.txt docs\\notes.txt
move draft.txt final.txt`}
/>

## Remove files or directories

<ShellCommandSet
  title="Remove content"
  unix={`rm notes.txt
rm -r old-folder`}
  powershell={`Remove-Item notes.txt
Remove-Item old-folder -Recurse`}
  cmd={`del notes.txt
rmdir /s old-folder`}
/>

<Warning title="rm deserves extra care">
  <p>Destructive terminal commands such as <code>rm</code>, <code>Remove-Item</code>, <code>del</code>, and <code>rmdir</code> deserve extra care. They often delete directly rather than sending content to a graphical trash folder. Read the path carefully before running them.</p>
</Warning>

## A practical rule

Creation and copying are usually reversible.

Removal is not always easy to undo.

That is why terminal users often move more slowly around removal commands than around creation, copying, or renaming.
