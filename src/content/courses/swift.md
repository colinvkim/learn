Swift is a compiled, type-safe language used for iOS, macOS, watchOS, tvOS, server tools, command-line programs, and more.

This course teaches Swift as a general-purpose programming language first, then applies that foundation to SwiftUI app development. The focus is practical: clear code, safe models, predictable data flow, modern concurrency, and small application structure.
