C++ is a compiled systems programming language built around explicit control. It is used where performance, memory layout, predictable resource management, and direct interaction with hardware or operating systems matter.

This course teaches modern C++ as a language of values, lifetimes, ownership, and resource management. You will use the standard library by default, learn why undefined behavior is dangerous, and build toward multi-file programs with clear ownership and practical performance habits.
